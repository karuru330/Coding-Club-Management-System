This project is developed for coding club team in our college that will provides interface to coding club team to 
manage coding competitions such as creating/updating/deleting coding competitions in offline unlike hackerrank and hackerearth.
students can participate in the coding competitions conducted by coding club team. students can run and submit their code and 
the code will be judged and result will be displayed. individual challenge leaderboard and overall contest leaderboard will be
updated time to time and displayed. Placement cell in the college can also use this software to conduct coding rounds in
recruitment drives.
